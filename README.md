## 🇬🇧 English Version

A professional binaural beat generator optimized for high-end headphones like the **Sony WH-1000XM5**. This web application helps users achieve different mental states through brainwave entrainment using the Web Audio API.

### 🚀 Live Demo
**👉 [Open the Binaural Lab here](https://lukasnikodem95-eng-cz.github.io/Binuar-Lab/)**

### ✨ Features
* **Full Responsive Design:** Optimized for desktop, iPhone, and Android.
* **Smart Fullscreen:** Visualization adapts its orientation based on your device (Portrait/Landscape).
* **5 Languages:** 🇨🇿 🇬🇧 🇩🇪 🇪🇸 🇫🇷
* **Neon Visualization:** Real-time interference wave display with neon effects.
* **Special Presets:** Fast access to Gamma, Beta, Alpha, Theta, and Delta states.

### 🎧 How to use
1. **Wear Headphones** (mandatory for the effect).
2. Choose a preset or set custom frequencies.
3. Use the **Swap** button to switch between Energy (L>R) and Relax (R>L) modes.
4. Click the **Visualization** to enter Fullscreen mode.

## 🇨🇿 Česká Verze

Profesionální generátor binaurálních tepů optimalizovaný pro špičková sluchátka jako **Sony WH-1000XM5**. Tato webová aplikace využívá Web Audio API k vytvoření frekvencí, které pomáhají mozku ladit se na specifické stavy vědomí.

### 🚀 Spustit aplikaci
**👉 [Vstoupit do Binaural Lab zde](https://lukasnikodem95-eng-cz.github.io/Binuar-Lab/)**

### ✨ Hlavní Funkce
* **Plně Responzivní:** Design vyladěný pro monitory i mobilní telefony (iPhone/Android).
* **Chytrý Fullscreen:** Vizualizace automaticky rozpozná orientaci displeje (na výšku pro mobil, na šířku pro PC).
* **Neonová Vizualizace:** Živé zobrazení sinusoid s efektem neonové záře.
* **5 Jazyků:** Podpora pro 🇨🇿 🇬🇧 🇩🇪 🇪🇸 🇫🇷
* **Vyladěné Presety:** Rychlá volba pro Spánek, Meditaci, Relaxaci, Fokus a Výkon.

### 🧠 Jak to funguje?
Binaurální tep vzniká, když do každého ucha pustíte mírně odlišnou frekvenci. Mozek tento rozdíl zpracuje a vytvoří "třetí tón", na který se následně naladí (např. rozdíl 10 Hz vyvolá stav Alpha). 
**Upozornění:** Pro správnou funkci **musíte použít sluchátka**.
